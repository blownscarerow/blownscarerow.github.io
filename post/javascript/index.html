• 定时器的介绍
• 定时器就是在一段特定的事件后执行某段程序代码
• 定时器的使用
• js定时器有两种创建方式
• setTimeout(func[delay,param1 ) 以指定的时间间隔调用函数的定时器
• setinterval(func[delay,param1 ) 以指定的时间间隔重复调用一个函数的定时器
• 第一个参数func 表示定时器需要执行的函数名
• 第二个参数 delay 时间间隔
• 第三个 是参数
